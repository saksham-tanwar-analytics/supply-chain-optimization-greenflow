# Supply Chain Optimization Model for Cost & Sustainability (Greenflow Cosmetics)

## Overview
Designed a multi-objective optimisation model to balance cost, lead time, and carbon emissions in supply chain planning.

## Business Problem
Supply chains require trade-offs between cost efficiency, delivery speed, and sustainability, which are difficult to optimise simultaneously.

## Solution
Developed a Python-based optimisation model with Tableau visualisations to analyse trade-offs and support strategic decision-making.

## Impact
Enabled data-driven supply chain planning by identifying optimal scenarios across cost, time, and environmental impact.

## Key Features
- Multi-objective optimisation model
- Trade-off analysis (cost vs lead time vs emissions)
- Scenario-based decision support
- Tableau visualisation for executive insights

## Tech Stack
Python · Tableau · Optimization (MILP) · pandas
